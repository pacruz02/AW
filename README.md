# ⚡ üNTER

**üNTER** es una aplicación web moderna diseñada para la gestión y reserva de vehículos eléctricos. El proyecto busca fomentar la movilidad sostenible mediante una experiencia de usuario fluida y un sistema de reservas eficiente.



---

## 🌟 Características Principales

* **Catálogo Dinámico:** Exploración de flota con filtros avanzados por autonomía, tipo de conector y precio.
* **Sistema de Reservas:** Calendario interactivo para gestionar fechas de alquiler.
* **Dashboard de Usuario:** Seguimiento de viajes, facturación.
